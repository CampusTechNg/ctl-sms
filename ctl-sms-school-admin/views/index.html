School Admin

Module Description

  • School Profile


    Create a profile for the school by providing the following information about the school; school name, school address, school phone number, school email, school website and school logo.

    Please note that the information provided on the school profile will be used by other modules to render some information. For example, the school name, logo and address will be displayed on students' result sheet.

  • Academic Sessions


    Create, edit and delete school academic sessions. Activating or deactivating an academic session can also be done here.

  • Academic Terms


    Create, edit and delete school academic terms. Activating or deactivating an academic term can also be done here.

  • Score Grades


    Manage the maximum score and remark for a grade. Example, grade - A, maximum score - 100, remark - excellent.

  • Set Scores


    Create a score template. Set the maximum score for continuous assessments and examination
